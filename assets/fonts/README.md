# Bundled fonts

The bundled fonts are stored locally so the font selector works without a network connection.

Google Fonts sources:

- Roboto: https://github.com/google/fonts/tree/main/ofl/roboto
- Inter: https://github.com/google/fonts/tree/main/ofl/inter
- Noto Sans: https://github.com/google/fonts/tree/main/ofl/notosans
- Noto Serif: https://github.com/google/fonts/tree/main/ofl/notoserif
- Open Sans: https://github.com/google/fonts/tree/main/ofl/opensans
- PT Sans: https://github.com/google/fonts/tree/main/ofl/ptsans
- PT Serif: https://github.com/google/fonts/tree/main/ofl/ptserif
- Ubuntu: https://github.com/google/fonts/tree/main/ufl/ubuntu
- Montserrat: https://github.com/google/fonts/tree/main/ofl/montserrat

OpenDyslexic source:

- https://forge.hackers.town/antijingoist/opendyslexic

The corresponding license text for each font is stored in `licenses/`.
